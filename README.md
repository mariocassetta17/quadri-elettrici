# ⚡ Global Electric Panel Builder (GEPB)

Il primo sistema al mondo per la progettazione grafica e la produzione automatizzata di quadri elettrici con logistica integrata e spedizione gratuita globale.

## 🚀 Visione
Diventare il punto di riferimento mondiale (come Amazon) per l'automazione industriale, permettendo a chiunque di costruire un quadro elettrico tramite interfaccia 3D.

## 🛠️ Architettura Tecnica
- **Frontend:** Configurato per drag & drop grafico dei componenti.
- **Backend:** Node.js con gestione sicura delle credenziali tramite variabili d'ambiente.
- **Database:** Struttura scalabile per milioni di componenti e schemi tecnici.

## 🔒 Sicurezza
Il progetto utilizza un'architettura a variabili d'ambiente per proteggere le chiavi API e le password del database durante lo sviluppo e il deployment.
